Introduction

This program handles secure encryption and decryption of data using AES and RSA. Key management is simplified with obfuscated key files.

Features

  • AES 256-bit Encryption
  • RSA 4096-bit Key Management
  • Obfuscation for Key Security
  • File Encryption Support
  • Key File Hashing for Integrity

Requirements

Python 3.x and cryptography library (install via pip).

Installation

pip install cryptography

Usage Guide

Key File Initialization

initialize_key_file("keyfile.key", "your-secure-password")

Encrypting Data

encrypt_data("keyfile.key", "your-password", b"Your sensitive data")

Decrypting Data

decrypt_data("keyfile.key", "your-password", "encrypted_data_string")

Encrypting Files

encrypt_file("keyfile.key", "your-password", "example.txt")

Decrypting Files

decrypt_file("keyfile.key", "your-password", "example.txt.enc")

Security Measures

  • RSA 4096-bit Key Protection
  • AES 256-bit Encryption
  • Key Obfuscation
  • File Hashing
  • Error Handling
